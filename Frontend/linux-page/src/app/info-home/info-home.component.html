Permisos UNIX
  • R (read) -> 4
  • W (write) -> 2
  • X (execute) -> 1

Archivos
  • - (archivo plano)
  • d (directorio)
  • l (enlace)
  • b (archivo de bloque)
  • c (archivo de caracteres)
  • p (archivo pipe)
  • s (archivo socket)

> ls -l

> - rwx r-- r-- 2 root root 320 2012-03-12 file.txt


  • - ← tipo de archivo
  • rwx ← permisos de propietario
  • r-- ← permisos de grupo
  • r-- ← permisos de otros
  • 2 ← numero de hardlinks
  • root ← dueño del archivo
  • root ← grupo del archivo
  • 320 ← tamaño del archivo en bytes
  • 2012-03-12 ← fecha y hora de la última modificación del archivo
  • file.txt ← nombre del archivo
